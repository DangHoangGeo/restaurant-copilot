# 🍽️ Shop-Copilot – Mobile-First SaaS for Small Restaurants

Shop-Copilot is a modern, mobile-first SaaS platform that helps small restaurant owners manage their entire operation from their smartphones. It offers a responsive admin dashboard, a customizable customer-facing website, and a native iOS app to streamline kitchen and checkout workflows.

## 🚀 Features

### 🔧 For Restaurant Owners (Admin Web App)
- Register and configure your restaurant
- Custom subdomain (e.g., `myrestaurant.yourservice.com`)
- Manage menu, categories, pricing, and availability
- Create and manage tables and employee shifts
- View real-time orders and sales reports
- Customize frontend website with your brand

### 🍽️ For Customers
- Scan table QR code to browse menu and order
- Each session is unique and expires after checkout
- Seamless mobile ordering experience

### 📱 For Restaurant Staff (iOS App)
- Real-time order notifications from the website
- Auto-print to Bluetooth kitchen printer (ESC/POS)
- Visual kitchen display to group similar items
- Checkout and receipt printing
- Designed for speed, clarity, and minimal manual input

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router), Tailwind CSS, TypeScript
- **Backend:** Supabase (Auth, Database, Storage, Realtime)
- **Mobile App:** Swift (iOS Native)
- **Deployment:** Vercel

## 🌐 Architecture Highlights

- Multi-tenant architecture with unique subdomains
- Realtime syncing between customer orders and staff app
- QR-based session ordering per table
- Automatic printing and digital kitchen board
- Admin panel with full restaurant control
